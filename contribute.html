Contribute

Entry Format

Add entries to the ENTRIES array in js/data.js:

{
  title: "Tool Name - What It Does",
  description: "Brief description",
  category: ["Memory"],
  phase: ["Analysis"],
  os: ["Linux", "Windows"],
  tools: ["ToolName"],
  commands: [
    { label: "Basic usage", cmd: "tool -f {DUMP_FILE} --option" }
  ],
  references: ["https://example.com/docs"]
}

Valid Values

Category

Crypto/Encoding Disk File Analysis Logs Malware Memory Metadata Network Registry Stego References

Phase

Acquisition Analysis Carving Recovery Timeline Triage

OS

Linux Windows macOS Android iOS

Placeholders

  • {DUMP_FILE} — Memory dump path
  • {IMAGE} — Disk image path
  • {PCAP} — Packet capture path
  • {TARGET_IP} — Target IP
  • {FILE} — Generic file path

Submit

git clone https://github.com/thureinoo/4n6.git
git checkout -b add-new-tool
# edit js/data.js
git commit -am "Add: ToolName - brief description"
git push origin add-new-tool

Then open a pull request on GitHub.

Keep commands copy-paste ready. Test before submitting. One entry per tool.
